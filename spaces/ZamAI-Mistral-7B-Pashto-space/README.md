# ZamAI-Mistral-7B-Pashto Space

This is a Space for the ZamAI-Mistral-7B-Pashto model. You can:

1. Test the model
2. Train the model
3. Fine-tune the model

Uses ZeroGPU for efficient GPU acceleration.

**Input:**
```
سلام، څنګه یی؟
```

**Output:**
